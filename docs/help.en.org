* Bank Statement Analyzer – Help

(under construction)

** Overview
This app allows for importing CSV bank statements (currently from two Polish banks, PKO BP and mBank), categorizing transactions (assigning them to spending categories), and generating a spending report.

An important feature of the app is that it works fully locally – no Internet connection is required for it to work, no data is sent anywhere, no account/registration is needed to use it.

A typical workflow looks like this:
- create spending categories
- create /rules/ ("/if/ a transaction's description contains a /string/, /then/ it belongs to /category/")
- import a CSV file (which categorizes all transactions according to the created rules)
- view and/or CSV-export a spending report

You can run ~demo.bat~ (Windows) or use the ~--demo~ CLI argument to run in /demo mode/.  All data in demo mode are deleted after quitting the app, but you can see a few categories, rules, and transactions to get a feeling of how to use the app.

Visit the app's GitHub repo at https://github.com/mbork/bank-statement-analyzer to learn more, report bugs, or contribute.
